This application is an advanced 3D encryption engine powered by Chaos Theory and the Butterfly Effect. Unlike standard algorithms, it utilizes Edward Lorenz's non-linear differential equations to transform messages into unpredictable mathematical chaos.

## ✨ Features
* **Chaotic Cryptography:** Uses the highly sensitive Lorenz attractor ($X_0, Y_0, Z_0$) to generate unique encryption keys.
* **3D Visualization:** Real-time generation of the "Strange Attractor" using Plotly.
* **Interactive Lab:** Adjustable chaos density (iterations), line thickness, and dynamic color themes.
* **Session Management:** Randomize keys or reset to default settings seamlessly.

## 🛠️ Tech Stack
* **Python**
* **Streamlit** (Web Framework & UI)
* **Plotly Graph Objects** (3D Rendering)
* **NumPy** (Mathematical computing)

## 🚀 How to Run Locally
1. Clone this repository.
2. Install the required libraries:
   `pip install -r requirements.txt`
3. Start the Streamlit server:
   `streamlit run app.py`
